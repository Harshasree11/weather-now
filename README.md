#  Weather Now

A simple weather application that lets users check the current and forecast weather for any city using the **Open-Meteo API**.

##  Features
- Current weather conditions (temperature, wind, feels like)
- 3-day forecast with icons
- Hourly weather (next 12 hours)
- Search any city globally

##  Tech Stack
- React.js (Frontend)
- Open-Meteo API (Weather Data)
- Open-Meteo Geocoding API (City Coordinates)

##  Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/weather-now.git
2. Install dependencies:
